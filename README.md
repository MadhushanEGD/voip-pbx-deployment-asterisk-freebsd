# VoIP PBX Deployment using Asterisk and FreeBSD

## Overview

This project demonstrates the deployment and configuration of a VoIP PBX system using Asterisk on FreeBSD.

The system supports SIP endpoint registration, voice calling, voicemail services, call routing, and call detail record management.

## Technologies Used

- FreeBSD
- Asterisk
- PJSIP
- MicroSIP
- Linphone
- VirtualBox
- Wireshark

## Features

- SIP Endpoint Registration
- Voice Calling
- Voicemail Configuration
- Call Routing
- Call Detail Records (CDR)
- SIP and RTP Traffic Analysis

Completed as part of Voice and Video over IP practical work.
